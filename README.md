## How to get Kuvio 

1. **Repository klonen**

`git clone https://gitlab.ai.it.hs-worms.de/ema-ss25/team_3.git`

`cd team_3`

2. **Abhängigkeiten installieren**

`flutter pub get`

3. **Firebase** 

Die Datei google-services.json für Android ist bereits im Repo, eine zusätzliche Einrichtung ist nicht nötig.

4. **App starten**

`flutter run`
